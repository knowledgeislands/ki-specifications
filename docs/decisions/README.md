# Decision Records - KI Specifications

This directory holds the significant, durable decisions for KI Specifications. Records use the Knowledge Islands Decision Records format and are living present-state records: update the record when the decision changes, rather than adding a supersession chain.

## Reading order

1. [GDR-KI-SPECIFICATIONS-001](GDR-KI-SPECIFICATIONS-001-adopting-decision-records.md) - adopts Decision Records for this repository.
2. [GDR-KI-ARCADIA-002](GDR-KI-ARCADIA-002-knowledge-islands-ecosystem-fundamentals.md) - defines the four repository authorities, publication flows, and choreography.
