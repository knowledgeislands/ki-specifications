# Specifications

This directory is the KIS registry: the accepted, normative specifications of the Knowledge Islands ecosystem.

A **KIS** (Knowledge Islands Specification) is the normative document produced once a KIP (Knowledge Islands Proposal) is accepted; see [../README.md](../README.md) for how the two relate, and [../docs/specification-process.md](../docs/specification-process.md) for the full lifecycle.

For how a KIS is drafted, numbered, and promoted from `Draft` to `Active`, and how its versions relate to schema and package versions, see [../docs/specification-process.md](../docs/specification-process.md) and [../docs/versioning.md](../docs/versioning.md).
